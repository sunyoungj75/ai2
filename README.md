# 가열 곡선 탐구 실험실 (State-Change Heating Curve Lab)

중학교 1학년 과학 수업용 가상 실험 웹앱입니다. -20°C 얼음을 가열해 수증기가 될 때까지의 온도 변화를 관찰하며, 학생들이 스스로 녹는점·끓는점(상태 변화 중 온도가 일정하게 유지되는 구간)을 찾아내도록 설계된 탐구형 시뮬레이션입니다.

## 주요 기능

- **가상 실험**: 물의 양(50~200g), 압력(0.5~3 atm)을 조절하며 가열 곡선을 실시간으로 관찰
- **미시적 관찰**: 비커 안 얼음/물/수증기의 입자 움직임과 배열을 함께 시각화
- **예측 그리기**: 그래프 위에 마우스로 직접 예측선을 그려본 뒤 실제 곡선과 비교
- **이전 실험 겹쳐보기**: 질량·압력을 바꿔가며 여러 번 실험한 결과를 겹쳐 비교
- **호버 재생**: 실험이 끝난 뒤 그래프 위에 마우스를 올리면 그 순간의 입자 배열을 다시 확인
- **탐구 질문**: 예측 → 발견 → 질문 → 비교 → 설명으로 이어지는 6개의 서술형 질문
- **학생 모드 / 교사 모드**: 학번·이름 확인 후 탐구 결과를 제출하고, 교사는 비밀번호로 접속해 제출 현황을 확인·구글 시트로 내보내기 가능

## ⚠️ GitHub Pages 등 일반 호스팅에 배포할 때 반드시 알아두세요

이 앱의 **학생 제출 데이터 수집 및 교사 모드 대시보드 기능**은 `window.storage`라는 API를 사용합니다. 이 API는 **Claude.ai 아티팩트(Artifact) 환경에서만 제공되는 기능**이며, 일반 웹 브라우저나 GitHub Pages 같은 정적 호스팅에서는 **존재하지 않습니다.**

즉, 이 HTML 파일을 GitHub Pages 등에 그대로 배포하면:

| 기능 | GitHub Pages 배포 시 |
|---|---|
| 가상 실험 (그래프, 입자 애니메이션, 예측선 등) | ✅ 정상 작동 |
| 학번/이름 확인 후 입장 | ✅ 정상 작동 (브라우저 내 검증만 사용) |
| 탐구 질문 답변 작성 | ✅ 정상 작동 |
| **"선생님께 제출하기" 버튼** | ❌ 오류 발생 (window.storage 없음) |
| **교사 모드 제출 현황 조회** | ❌ 항상 빈 목록으로 표시됨 |

**권장 방법**: 학생 데이터 수집 기능까지 온전히 사용하려면, GitHub이 아니라 **Claude 아티팩트의 "게시(Publish)" 기능**으로 공개 링크를 만들어 배포해주세요. 이 경우 모든 기능이 정상 작동합니다.

GitHub Pages는 다음과 같은 경우에 적합합니다:
- 데이터 제출 기능 없이 **가상 실험 도구로만** 활용하고 싶은 경우
- 직접 백엔드(Firebase, Google Apps Script 등)를 연결해 제출 기능을 다시 구현할 여력이 있는 경우

## GitHub Pages 배포 방법 (실험 도구로만 사용 시)

1. 이 저장소를 GitHub에 업로드 (또는 새 저장소 생성 후 `heating_curve_lab.html` 파일 추가)
2. 저장소 **Settings > Pages** 이동
3. **Source**를 `main` 브랜치, `/ (root)` 폴더로 설정 후 저장
4. 잠시 후 `https://<사용자명>.github.io/<저장소명>/heating_curve_lab.html` 주소로 접속 가능
5. 필요하면 `index.html`로 파일명을 바꾸면 저장소 루트 주소로 바로 접속 가능
